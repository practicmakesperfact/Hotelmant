# Leul Mekonen Hotel Management System

A comprehensive, modern Hotel Management System built for **Leul Mekonen Hotel** in Kombolcha, Ethiopia. This application provides a seamless experience for guests, staff, and management, blending modern luxury with traditional Ethiopian hospitality.

## 🏨 Features

### 🌟 For Guests
- **Dynamic Booking System**: Easy-to-use interface for selecting and booking various room types.
- **Room Showcase**: Stunning visuals and detailed descriptions of Standard, Deluxe, Suite, and Presidential rooms.
- **User Dashboard**: Manage profiles, view booking history, and handle cancellations.
- **Multi-language Support**: Experience the platform in multiple languages (English and Amharic).

### 🛠️ For Operations (Staff & Management)
- **Receptionist Portal**: Manage check-ins, check-outs, and guest requests.
- **Housekeeping Tracking**: Real-time status updates for room cleaning and maintenance issues.
- **Inventory Management**: Track hotel supplies and equipment efficiently.
- **Manager Reports**: Visualized data and analytics for hotel performance oversight.
- **Admin Dashboard**: Comprehensive control over staff, rooms, and system settings.

## 🚀 Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Forms & Validation**: [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)
- **Data Visualization**: [Recharts](https://recharts.org/)
- **State Management**: React Context & Hooks

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18.x or higher)
- npm, pnpm, or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/practicmakesperfact/hotelMant.git
   cd hotelMant
   ```

2. **Install dependencies**:
   ```bash
   npm install
   # or
   pnpm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open the application**:
   Navigate to [http://localhost:3000](http://localhost:3000) to see the result.
